Parte III¶

Uma indstria dispes de 3 recursos, matria-prima, mo de obra e tempo de maquinrio, em quantidades limitadas. Com esses recursos, a indstria pretende produzir dois produtos, 1 e 2. O quadro d a utilizao de cada recurso em cada um dos produtos e a disponibilidade de cada recurso.

A indstria sabe que cada unidade produzida do produto A d um lucro de 5,00 Reais e do produto B um lucro de 6,00 Reais.

Premissa: Maximizar o lucro

Processo de Modelagem¶

1) Variáveis de Decisão:¶

Generalizado: Varivel de Deciso:

  • $x_i$: Quantidade unitria do produto i a ser produzida

ndices

  • $i$: Produto do tipo i
  • $j$: Recurso j consumido para produzir uma unidade do produto do tipo i

Extenso dos Conjuntos dos ndices

  • $n$: Tipos diferentes de produtos considerados
  • $m$: Quantidade de recursos necessrios para produzir os produto considerados

Aplicado ao Problema:

  • $x_1$: Quantidade unitria do produto 1 a ser produzida
  • $x_2$: Quantidade unitria do produto 2 a ser produzida

2) Parâmetros¶

Consumo de Recursos

Generalizado

  • $CONSUMO_{ij}$: consumo em unitrio do recurso j para a produo de uma unidade do produto i

Aplicado ao Problema

  • $CONSUMO_{11}$: consumo unitrio do recurso 1 para produzir uma unidade do produto 1
  • $CONSUMO_{12}$: consumo unitrio do recurso 2 para produzir uma unidade do produto 1
  • $CONSUMO_{13}$: consumo unitrio do recurso 3 para produzir uma unidade do produto 1
  • $CONSUMO_{21}$: consumo unitrio do recurso 1 para produzir uma unidade do produto 2
  • $CONSUMO_{22}$: consumo unitrio do recurso 2 para produzir uma unidade do produto 2
  • $CONSUMO_{23}$: consumo unitrio do recurso 3 para produzir uma unidade do produto 2

Preos Generalizado

  • $LUCRO_i$: lucro por venda de uma unidade do produto $i$

Aplicado ao Problema

  • $LUCRO_1$: lucro por venda de uma unidade do produto 1, R\$ 5,00/un
  • $LUCRO_2$: lucro por venda de uma unidade do produto 2, R\$ 6,00/un

Disponibilidade de Recursos Generalizado

  • $DISPONIBILIDADE_j$: disponibilidade do recurso j

Aplicado ao Problema

  • $DISPONIBILIDADE_1$: disponibilidade de matria-prima, 14 un
  • $DISPONIBILIDADE_2$: disponibildiade de mo-de-obra, 9 un
  • $DISPONIBILIDADE_3$: disponibildiade de tempo de maquinrio, 56 un

*IMPORTANTE: Daqui em diante a modelagem ser apresentada na forma generalizada devido extenso dos problemas.

3) Função Objetivo¶

Objetivo: Maximizar o lucro

Funo Objetivo - Generalizada:

$$ max \, Z = \sum_{i = 1} ^{n} LUCRO_i x_i $$

4) Restrições¶

Restries Generalizadas

1) Restrio de disponibilidade do recurso j por ms

\begin{gathered} \begin{aligned} &\sum_{i=1}^{n}\sum_{j=1}^{m} CONSUMO_{ij} x_i \leq DISPONIBILIDADE_j & \end{aligned} \end{gathered}

2) Restrio de no-negatividade

\begin{aligned} &\ x_i \geq 0 \end{aligned}

5) Formulação Matemática Formal¶

$\displaystyle max \, Z = \sum_{i = 1} ^{n} LUCRO_i x_i $

$\displaystyle s.t. $

$\displaystyle\sum_{i=1}^{n}\sum_{j=1}^{m} CONSUMO_{ij} x_i \leq DISPONIBILIDADE_j $ $\,\,\,$ $\forall$ i $=1,...,n,$ $\,$ $\forall$ j $=1,...,m $

$\displaystyle\ x_i \geq 0$ $\,\,\,$ $\forall$ i $=1,...,n$

# Importando gurobipy API
import gurobipy as gp

# Criando dados de recursos e parmetros associados
recurso, disponibilidade = gp.multidict({
    "Matria-prima": 14,
    "Mo-de-obra": 9,
    "Tempo de Maquinrio": 56})

# Criando dados de lucro por tipo rao
tipoProduto, lucro = gp.multidict({
    "Tipo1": 5,
    "Tipo2": 6 })

# Criando relao de recursos demandados por tipo de produto
consumoRecursos = {
    ("Matria-prima", "Tipo1"): 1,
    ("Matria-prima", "Tipo2"): 2,
    ("Mo-de-obra", "Tipo1"): 1,
    ("Mo-de-obra", "Tipo2"):  1,
    ("Tempo de Maquinrio", "Tipo1"): 7,
    ("Tempo de Maquinrio", "Tipo2"):  4}

# Criando o modelo
m = gp.Model("Max Lucro Produo")

# Criando variveis de deciso
Qt_tipoProd = m.addVars(tipoProduto, name = "Tipo de Produto")
